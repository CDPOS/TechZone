# ⚡ TechZone - Hardware & Periféricos Gamer

> 🚀 **Veja o projeto online:** [https://cdpos.github.io/TechZone/]

Este projeto é o marco inicial (Etapa 1) de um site de e-commerce focado em componentes de alta performance e periféricos para o público gamer e entusiastas de tecnologia.

## 🎯 Público-Alvo
O site foi desenvolvido para:
* Gamers que buscam upgrades de hardware (Processadores, Memórias).
* Profissionais de tecnologia que necessitam de periféricos ergonômicos e precisos.
* Entusiastas de hardware que valorizam especificações técnicas detalhadas.

## 🗺️ Mapa de Páginas
O projeto é composto por 4 páginas principais:
1.  **Home (`index.html`)**: Apresentação da loja, banner promocional e vitrine com as principais ofertas.
2.  **Produtos (`produtos.html`)**: Catálogo completo utilizando estrutura de listas semânticas.
3.  **Contato (`contato.html`)**: Formulário funcional com validações de campo, seleção de produtos e níveis de urgência.
4.  **Especificações (`especificacoes.html`)**: Esta página é dinâmica e carrega os detalhes técnicos de cada produto usando JavaScript (via parâmetros na URL).

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estrutura semântica completa (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`).
* **CSS3**: Estilização moderna com **CSS Grid** (layout responsivo) e **Flexbox**.
* **JavaScript**: Lógica para carregamento dinâmico de dados e validações de formulário em tempo real.

## Como Executar o Projeto
1.  Faça o download ou clone este repositório.
2.  Certifique-se de manter a estrutura de pastas:
    * Raiz: Arquivos `.html` e `README.md`.
    * Pasta `/css`: Arquivo `estilos.css`.
3.  Abra o arquivo `index.html` em qualquer navegador moderno (Chrome, Edge, Firefox).
