⚡ TechZone - Hardware & Periféricos Gamer
🚀 Veja o projeto online: [https://cdpos.github.io/TechZone/]

Este projeto é uma plataforma de e-commerce completa e responsiva, focada em componentes de alta performance e periféricos gamer. O site combina uma interface moderna "Dark Theme" com uma experiência de usuário (UX) fluida e funcionalidades inteligentes de front-end.

🎯 Público-Alvo
Gamers & Streamers: Que buscam o que há de mais moderno em processamento e periféricos RGB.

Entusiastas de Hardware: Que valorizam especificações técnicas e comparação de preços.

Profissionais de Tecnologia: Que buscam ergonomia e performance para o setup de trabalho.

🗺️ Mapa de Páginas
O projeto é composto por 5 páginas interconectadas:

Home (index.html): Banner promocional e acesso rápido às ofertas por ordenação de preço.

Produtos (produtos.html): Catálogo completo com sistema de filtro por categoria e ordenação simultânea.

Carrinho (carrinho.html): Gerenciamento de itens selecionados, controle de quantidade (limite de 10 unidades) e cálculo de subtotal em tempo real.

Contato (contato.html): Formulário avançado com máscara de telefone, trava de idade mínima (10 anos) e seletor de urgência.

Especificações (especificacoes.html): Visualização detalhada e dinâmica de cada produto baseada em banco de dados JS.

🛠️ Tecnologias e Recursos
HTML5 Semântico: Uso rigoroso de tags como <article>, <section> e <nav> para SEO e acessibilidade.

CSS3: Layout totalmente responsivo utilizando CSS Grid e Flexbox, com estilização customizada de componentes nativos (Input Range).

JavaScript:

Banco de dados local para persistência do carrinho via LocalStorage.

Lógica de filtragem e ordenação cruzada.

Validações dinâmicas de formulário (Data de nascimento e máscaras).

SweetAlert2: Biblioteca externa para modais e alertas elegantes, substituindo os diálogos nativos do navegador.

🚀 Funcionalidades de Destaque
Sistema de Carrinho: Adição, remoção e alteração de quantidade com feedback visual no ícone do menu.

Trava de Segurança: O sistema impede cadastros de usuários com menos de 10 anos, calculando a idade dinamicamente.

Filtros Inteligentes: Permite filtrar por categoria (ex: Hardware) e ordenar por preço ao mesmo tempo, sem perder o estado da busca.

Como Executar o Projeto
Clone este repositório.

Mantenha a estrutura de pastas:

/css: estilos.css.

/js: app.js.

/img: Imagens dos produtos em formato .webp.

Abra o index.html em seu navegador de preferência.
